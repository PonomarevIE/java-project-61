### Five console games to improve your mind. The games are played in question-answer mode. Three rounds. The player wins if he answered all three questions correctly.

### Start
```bash
make
```

### Setup
```bash
make build
```

### Install
```bash
make install 
```

### Run
```bash
make run-dist
```

### Run checkstyle
```bash
make lint
```
